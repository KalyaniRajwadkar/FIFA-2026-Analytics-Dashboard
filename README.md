## 🎯 Project Goal

Modern football is driven by data.

The objective of this project is to build an advanced FIFA 2026 Analytics Dashboard capable of transforming football data into actionable insights through interactive visualizations and analytics.

The dashboard will help analyze:

- Team Performance
- Player Statistics
- Goals Distribution
- Shot Analysis
- Match Insights
- Tournament Trends
- Performance Prediction
